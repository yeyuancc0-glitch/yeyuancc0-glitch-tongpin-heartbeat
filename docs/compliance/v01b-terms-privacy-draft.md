# V0.1B 用户协议与隐私政策草案

## 用户协议草案要点

- 本产品是情侣双人私密空间，不提供公开社区或陌生人匹配。
- 用户应只上传、书写自己有权分享的内容。
- 用户可以删除自己创建的留言、相册图片、信件和今日胶囊。
- 拉黑当前伴侣会结束当前情侣关系，双方不能继续写入原情侣空间。
- 举报会记录处理线索，完整审核后台和处理流程将在后续版本补齐。

## 隐私政策草案要点

- 采集账号邮箱、昵称、生日、头像、情侣关系、日常胶囊、留言、日历事件、相册图片、信件和站内通知数据。
- 情侣业务数据以 `couple_id` 做权限边界，只允许当前 active couple member 访问。
- 头像和相册存储在 Supabase private Storage bucket，通过临时 signed URL 展示。
- 数据库不保存 signed URL，不保存公开图片地址。
- 账号注销在 V0.1B 为最小闭环：记录注销申请、标记 profile 待注销、结束当前关系；物理删除账号和全量数据清理留待后续后台流程。

## 待 V1.0 补齐

- 正式法律文本。
- App 内完整账号删除入口和处理 SLA。
- 举报、屏蔽、内容处理和用户支持流程。
- 应用商店隐私标签和审核材料。
