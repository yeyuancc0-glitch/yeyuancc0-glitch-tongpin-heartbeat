# 「同频跳动」轻量化浪漫 UI 视觉重塑方案 (V2 优化定稿版 - 尊重核心资产)

本方案作为定稿版本，旨在凝聚视觉张力，并在重塑中**严格保护你的两项核心产品设计**：
1. **「品牌胶囊 (CapsuleMark)」**：作为你独特的品牌灵魂与创意暗喻，本方案对胶囊标识、胶囊文案体系以及“情绪胶囊”的封存逻辑进行**完全保留，不作任何改变**，并将其作为整个情感视觉设计的核心图腾。
2. **「底部导航栏 (Bottom Tab Bar)」**：当前优雅的半透明玻璃舱设计、液滴高光追踪、以及针对 iOS Safari 滚动阻尼的 `visualViewport` 适配技术具有极高的原生质感，本方案**完全保留导航栏的样式与结构，不作任何修改**。

> [!NOTE]
> **优化焦点**：在保留胶囊与导航栏的基础上，我们集中力气攻克首页卡片的“AI排版感”与“网页扁平化”问题。利用**三维深度空间**、**偏角错落物理模型**和**拟纸质手写温度**，在不实施任何代码的前提下，为应用输出一套极致浪漫、克制、且高度符合 App 原生规范的视觉设计方案。

---

## 🛠️ 重构原则：克制浪漫，告别“AI工具冷漠感”

*   **克制不堆叠**：不增加任何多余的装饰性彩条或花哨动效，仅通过**卡片阴影倾向**、**界面呼吸底色**与**文字排版呼吸感**来表达浪漫。
*   **物理手作感**：利用略带倾角的拍立得相框拼接、信纸折角与写实手绘线条，让情侣空间产生“两个人共同DIY的房间”的指尖触感。
*   **原生App规范**：摒弃网页扁平拼接，卡片全部采用 Squircle（超圆角）及微光内描边。

## 🎨 重塑方案细则 (Homepage Redesign Proposal)

### 1. 全局三维空间层级 (App-Native Canvas)

为了彻底打破网页的扁平感，我们将首页的视觉维度重置为三层物理透视空间：

*   **Layer 1 (Background - 呼吸底色)**：
    *   背景海浪般渐变。采用**极低饱和度的温和双色呼吸渐变**（从奶油粉 `#FFF3F6` 到暖香草黄 `#FFFAF0`）。
    *   背景交界处进行径向柔焦（Radial Blur）处理。滑动屏幕时，背景色块会产生极其微弱的阻尼视差移动，让整个画布“活”起来。
*   **Layer 2 (Canvas - 情绪介质)**：
    *   首页所有功能卡片统一升级为 **Squircle (超圆角 - iOS 1:1.2 标准圆角模型，`borderRadius: 28-32` )**。
    *   卡片材质为 **半透果冻玻璃态 (Translucent Jelly Glass)**：`backgroundColor: "rgba(255, 255, 255, 0.76)"`，带有极其细腻的白色内描边 `borderWidth: 1, borderColor: "rgba(255, 255, 255, 0.85)"`。
    *   **蔷薇雾影 (Mauve Soft Shadow)**：摒弃灰黑色投影，改用带暖粉倾向的超轻柔微影：`shadowColor: "#5A444C"`, `shadowOffset: { width: 0, height: 10 }`, `shadowOpacity: 0.04`, `shadowRadius: 20`，卡片如同温和地浮于云朵之上。

### 2. 首页核心模块重塑方案 (Homepage Components)

#### A. 恋爱纪念主卡 (Love Days Card) ——「同频共振线」
恋爱天数是情侣的专属纪念碑，视觉上应兼具现代排版的优雅与情感的跳动。

*   **视觉方案**：
    *   **心电共振曲线 (Heartbeat Wave)**：在 Me 与 Partner 头像之间，将现有的硬接线升级为一条**动态心电共振曲线**（SVG 渐变路径）。曲线配合心率频率缓缓产生微小的正弦波澜，象征两颗心脏的“同频跳动”。
    *   **editorial 叙事字形**：恋爱天数数字大字号化（`56pt`），字重选用优雅的中细体（Light-Medium），字体采用带有手作温度的经典衬线体（Serif Font，如 Outfit 或 Playfair），并在数字下方辅以浅蔷薇色微型印章，强化纪念仪式感。
    *   **「火漆封印」信封入口 (Love Letter Entry)**：
        *   完全遵守“位于开始日期下方、主卡内部首屏可见”的规则。
        *   将“写情书”入口重绘为一个**扁平且带有真实折叠阴影的模拟信封**。信封封口处点缀一枚半透明、果冻粉金色的**爱心火漆印章**。
        *   点击火漆印章时，火漆印章微缩回弹，以弹性信纸展开动画拉开写信界面，将冰冷的按钮点击转化为浪漫的“拆信仪式”。

#### B. 此刻同频卡 (Synchronous Heartbeat Card) ——「心情贴纸盒」
该区域承载着情侣每日的快速互动与心情同步。我们重点优化按钮质感，使其更具“指尖可触感”。

*   **视觉方案**：
    *   **双子气泡插槽 (Floating Mood Slots)**：
        *   “我的心情”（包含已存下的今日胶囊）与“TA的心情”合并为两个并排的**云朵充气气泡卡片**。
        *   已存下心情时，气泡背景呈现柔和的彩虹微渐变色，带有一丝从内透光的晶莹感；未存下时，呈现为温暖奶油白的虚线描边空腔，静默提示“等一封回应”。
    *   **微 3D 拟真互动贴纸 (Tactile Beveled Stickers)**：
        *   8 个快捷互动按钮升级为 **Squircle Rounded Stickers (圆角贴纸形)**。
        *   添加极其微弱的斜角边缘（Bevel Edge）与表层高光漫反射（Matte Reflection），使其在视觉上呈现出像真实 PVC 软胶贴纸一样的厚度与指尖按压反馈（Bouncy Press）。
        *   自定义互动按钮统一使用优雅的**手绘灰色虚线边框**与一个小小的毛茸茸加号（+），完美融入整体格网。
    *   **发射物理动效 (Parabolic Fly)**：
        *   点击贴纸时，该贴纸化作一颗小爱心沿着弧形物理抛物线飞向屏幕上方的对方头像，撞击后爆开一圈温柔的水波涟漪。

#### C. 共享相册卡 (Shared Album) ——「拍立得时光墙」
摒弃网页常用的生硬九宫格列表，改用能唤起浪漫回忆的纸质拼接。

*   **视觉方案**：
    *   **拍立得微偏角拼接 (Polaroid Collage)**：
        *   照片外层套上一圈微带厚度感的白色拍立得纸框，底部宽边处用精致的手写斜体（如 Courier 或 Script Font）印刷上上传人昵称与上传日期。
        *   照片卡片在网格布局中带有**极其微弱的随机旋转偏角**（例如：`-1.2°`、`+1.8°`、`-0.5°`），模拟真实的双人时光照片墙，自然温馨。
    *   **空白记忆底卡 (The Empty Memoir)**：
        *   相册未满时，上传入口呈现为一张**空白的拍立得相机胶片**，上面用淡淡的铅笔手绘线条画着一个相机，配文：“在这里，留存我们的下一个瞬间”。
    *   **共享飞行展开 (Shared Element Transition)**：
        *   点击缩略图查看大图时，照片卡片平滑地从当前偏角回正、变大并飞到屏幕中央，周围透出柔和的高斯模糊背景。

#### D. 日记留言板 (Message Board) ——「手写格纹信笺」
让打字留言的操作变成纸张书写的自然流露。

*   **视觉方案**：
    *   **浅粉格纹信纸 (Ruled Letter Pad)**：
        *   留言区背景改用极其淡雅的**粉白相间信纸格纹线**。
        *   卡片右上角刻画一个带有真实物理投影的**折角效果 (Paper Fold)**，增加立体拟真感。
    *   **纸面无框输入 (Underlined Pen Writing)**：
        *   取消突兀的矩形输入框，输入栏变为几行**淡淡的水平横线**。用户在横线上打字，文字如同用水彩笔写在信纸上一样。
    *   **手账便利贴墙 (Layered Memos)**：
        *   留言历史记录改用一个个**微重叠的便签卡片 (Layered Sticky Memos)**。
        *   卡片上方设计一个手绘回形针或胶带纸的图形，仿佛是一本两人共同手绘的情感手账本。

## 🚀 悬浮小窝入口设计 (Co-Creation Floating Entry)

为严格遵循“首页右下角悬浮圆形按钮”与“不更改共创模块本身”的规范，我们仅对其**首页入口按钮**进行高质感微调：

*   **视觉方案**：
    *   **梦幻微光气泡球 (Tactile Crystal Sphere)**：
        *   该圆形悬浮按钮重绘为一颗**高折射透明水晶球**。水晶球内部隐隐透出两人共享云宠猫狗的微缩萌态剪影。
        *   水晶球外侧环绕着一圈柔和的呼吸微光（Breathing Aura），按钮本身具有自然的重力摇晃微动效。
        *   当小窝里有新动态（如对方刚刚喂了宠物或新增了浪漫足迹）时，水晶球上方会缓缓降落一颗亮晶晶的粉金色星光（代替冷冰冰、刺眼的红色数字未读红点），传递温馨的陪伴呼吸。

## 📐 页面视觉设计参数规范 (Design Spec)

| 视觉属性 | 精准参数规范 | 原生 App 感体现说明 |
| :--- | :--- | :--- |
| **功能卡片主圆角** | `borderRadus: 32` (Squircle 超圆角) | 极其平滑的曲线过渡，彻底消除网页盒子的生硬凌厉感。 |
| **小贴纸/气泡圆角** | `borderRadus: 20` | 与主卡片保持有机比例，保证小块控件的饱满感与圆润感。 |
| **全局侧边距** | `paddingHorizontal: 16` | 原生 App 黄金边距，确保内容紧凑且留有呼吸空间。 |
| **卡片间纵距** | `marginVertical: 14` | 恰到好处的留白，形成有节奏的视觉呼吸感。 |
| **卡片背景色** | `rgba(255, 255, 255, 0.76)` (带毛玻璃模糊) | 轻透感材质，与呼吸渐变背景自然融合。 |
| **卡片投影参数** | `shadowColor: "#5A444C"`, `shadowOpacity: 0.04`, `shadowRadius: 20` | 铺设粉金色超轻影，卡片仿佛温和地浮于粉色云雾之上。 |
